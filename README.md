##  📘 프로그램 및 기능 소개
#### 프로그램 소개
<p>
  react와 node.js를 사용하여 제작한 쇼핑몰입니다.
</p>
<br>

#### 기능 소개
- ##### 로그인 X
  - 상품 목록 
  - 상품 상세 보기
  - 상품 검색
  - 로그인 및 회원 가입

- ##### 로그인 O
  - 상품 주문
  - 장바구니
  - 마이페이지(회원 정보 수정, 주문 내역 확인)
<br>
